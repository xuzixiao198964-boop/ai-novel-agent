## 2. 用户初始化选择流程

### 2.1 生成方式选择

#### 2.1.1 只生成小说模式
```
小说生成选项：
1. 篇幅选择 (必选)
   - 微型小说 (1-10章)
   - 短篇小说 (10-50章)
   - 中篇小说 (50-200章)
   - 长篇小说 (200-500章)

2. 题材选择 (必选)
   - 儿童故事
     ├─ 3-6岁童话
     ├─ 7-12岁故事
     └─ 13-18岁青少年
   
   - 男频分类
     ├─ 玄幻奇幻
     ├─ 军事战争
     ├─ 科幻未来
     ├─ 历史穿越
     └─ 都市异能
   
   - 女频分类
     ├─ 言情甜宠
     ├─ 霸总豪门
     ├─ 悬疑推理
     ├─ 纯爱耽美
     └─ 宫斗宅斗

3. 随机选项 (可选)
   - 篇幅随机
   - 题材随机
   - 完全随机 (系统自动决策)
```

#### 2.1.2 生成视频模式

##### 2.1.2.1 AI小说转视频
```
选项：
1. 小说来源
   - 新生成小说 (需选择篇幅和题材)
   - 已有小说 (从作品库选择)
   - 外部导入 (URL或文件上传)

2. 视频范围
   - 部分章节 (选择具体章节)
   - 全部章节
   - 随机选择

3. 视频生成选项
   - 语言生成画面 (AI生成视频画面)
   - 图片模式 (静态图片+语音)
   - 导入视频/音频 (用户自定义素材)
   - 背景音乐选择
   - 字幕选项 (样式、字体、颜色可调)
   - 语音选项 (语速、语调、音色可选)
```

##### 2.1.2.2 资讯转视频
```
选项：
1. 资讯来源
   - 原文版本 (原始语言)
   - 翻译中文版 (系统自动翻译)
   - 用户上传 (支持多种格式)

2. 内容处理
   - 大模型归纳总结
   - 口语化述说转换
   - 关键信息提取

3. 视频生成选项
   - 语音口型一致 (视频播报)
   - 背景音乐选择
   - 字幕选项
   - 语音选项
```

### 2.2 冲突处理规则
```
多选冲突处理规则：
1. 如果选项冲突，系统随机选择一种
2. 优先级：用户明确选择 > 系统随机选择
3. 冲突检测和提示机制
4. 用户确认或系统自动决策
```
