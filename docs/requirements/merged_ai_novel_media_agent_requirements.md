## 1. 项目概述

### 1.1 项目背景
将AI Novel Agent（AI小说生成系统）与Media Agent（资讯转视频系统）合并，构建一个统一的AI内容创作平台。用户可以选择生成小说或通过小说/资讯生成视频，支持多种视频生成模式选择。

### 1.2 核心功能
1. **AI小说生成**: 继承AI Novel Agent的7个Agent流水线，自动生成完整小说
2. **视频生成**: 继承Media Agent的视频处理能力，支持多种生成模式
3. **内容抓取**: 从网络抓取更多样化的内容作为创作素材
4. **篇幅选择**: 支持用户选择小说篇幅和视频时长
5. **成本测算**: 测算从小说生成到视频生成的完整流程时间和费用

### 1.3 技术架构
- **后端**: FastAPI + Python + DeepSeek LLM
- **数据库**: PostgreSQL + Redis
- **视频处理**: FFmpeg + 阿里云百炼VideoRetalk
- **TTS**: 腾讯云TTS + Edge TTS + OpenAI TTS
- **部署**: 服务器 104.244.90.202:9000

## 2. 功能需求

### 2.1 用户认证管理（继承Media Agent）
- **用户注册**: 用户名+密码，强密码策略
- **用户登录**: JWT令牌认证，会话管理
- **密码管理**: 修改、找回、过期策略
- **用户信息**: 显示名称、头像、个人简介

### 2.2 AI小说生成（继承AI Novel Agent）

#### 2.2.1 7个核心Agent流水线
1. **TrendAgent** - 热门趋势分析
   - 实时分析小说平台热门榜单
   - 动态生成题材、章节数、字数建议
   - 18种题材（含儿童），支持手动选择主题
   - 主题去重机制（3次内不重复）

2. **StyleAgent** - 风格解析
   - 分析目标小说的语言风格特征
   - 提取叙事节奏模式
   - 建立可量化的风格参数体系

3. **PlannerAgent** - 策划案生成
   - 故事总纲+章节大纲
   - 3章周期审核机制
   - 多轮审核优化
   - 测试模式隔离（覆盖章节数）

4. **WriterAgent** - 正文创作
   - 逐章正文创作
   - 前两章引子上下文保持一致性
   - 引用上一章后半段+故事总纲+策划案

5. **PolishAgent** - 章节润色
   - 语言优化和润色
   - 风格一致性检查

6. **AuditorAgent** - 质量审核
   - 质量审核打分
   - 多维评分体系

7. **ReviserAgent** - 修订优化
   - 根据审核结果修订
   - 优化内容质量

#### 2.2.2 小说篇幅选择
- **短篇小说**: 6-18章，每章2000-3000字
- **中篇小说**: 18-54章，每章2500-3500字  
- **长篇小说**: 54-162章，每章3000-4000字
- **自定义篇幅**: 用户自定义章节数和字数

#### 2.2.3 内容抓取扩展
- **小说平台**: 起点、晋江、纵横等
- **资讯平台**: 新闻、博客、社交媒体
- **内容类型**: 都市、玄幻、言情、科幻、历史等
- **抓取频率**: 每日更新题材库

### 2.3 视频生成（继承Media Agent）

#### 2.3.1 视频生成模式选择
1. **仅配音模式**
   - 小说/文案 + TTS语音合成
   - 静态背景图片或简单动画
   - 快速生成，成本最低

2. **仅字幕模式**
   - 小说/文案 + 字幕生成
   - 背景视频或图片轮播
   - 适合阅读类内容

3. **动画模式**
   - AI生成角色和场景
   - 动态动画效果
   - 适合儿童故事、奇幻题材

4. **混合模式**
   - 配音 + 字幕 + 动画
   - 完整的视频体验
   - 成本最高，效果最好

5. **资讯转视频模式**
   - RSS资讯抓取
   - AI摘要生成
   - 自动视频合成
   - 定时发布

#### 2.3.2 视频处理能力
- **视频合成**: FFmpeg本地处理
- **口型替换**: 阿里云百炼VideoRetalk
- **字幕生成**: 语音识别或文案同步
- **转场效果**: 淡入淡出、滑动、缩放等
- **背景音乐**: 内置音乐库或用户上传

#### 2.3.3 TTS语音合成
- **腾讯云TTS**: 高质量，支持音色克隆
- **Edge TTS**: 本地备选，无需API密钥
- **OpenAI TTS**: 备选方案
- **语音选择**: 根据语言自动选择，支持多种音色

### 2.4 内容抓取与素材库

#### 2.4.1 小说内容抓取
- **抓取源**: 主流小说平台
- **内容类型**: 章节内容、评论、评分
- **更新频率**: 每日更新热门题材
- **去重机制**: SHA-256哈希去重

#### 2.4.2 资讯内容抓取
- **RSS源管理**: 支持RSS 2.0和Atom 1.0
- **网页抓取**: Readability算法提取正文
- **图片处理**: 自动下载和优化
- **质量控制**: 内容评分算法

#### 2.4.3 素材库管理
- **图片素材**: 场景、角色、背景
- **音频素材**: 背景音乐、音效
- **视频素材**: 片段、转场效果
- **模板库**: 视频模板、小说模板

### 2.5 发布管理

#### 2.5.1 发布平台
- **小说平台**: 起点、晋江等（模拟发布）
- **视频平台**: 抖音、小红书、B站等
- **社交媒体**: 微博、微信公众号

#### 2.5.2 发布策略
- **立即发布**: 生成完成后立即发布
- **定时发布**: 指定时间发布
- **批量发布**: 多章节/多视频批量发布
- **审核发布**: 人工审核后发布

#### 2.5.3 发布监控
- **状态跟踪**: 实时监控发布状态
- **效果分析**: 观看量、点赞、评论等
- **失败重试**: 自动重试机制
- **报表生成**: 发布统计报表

  - 每章3000字: 约¥0.03
  - 短篇小说（6章）: 约¥0.18
  - 长篇小说（54章）: 约¥1.62

#### 4.1.3 视频处理成本
- **阿里云百炼**: 按分钟计费
- **口型替换估算**: 每分钟约¥0.5

### 4.2 服务器成本

#### 4.2.1 基础服务器成本
- **VPS租用**: 约¥100-200/月
- **带宽成本**: 按流量计费
- **存储成本**: 按容量计费

#### 4.2.2 任务处理成本
- **CPU时间成本**: 按处理时间计算
- **内存使用成本**: 按使用量计算
- **存储IO成本**: 按读写量计算

### 4.3 完整流程成本测算

#### 4.3.1 小说生成到视频生成完整流程
```
示例：生成54章小说并转为视频

1. 小说生成成本：
   - DeepSeek API: ¥4.5
   - 服务器处理: ¥2.0
   - 小计: ¥6.5

2. 视频生成成本：
   - TTS合成: ¥1.62
   - 视频处理: ¥5.0
   - 服务器处理: ¥3.0
   - 小计: ¥9.62

3. 总成本：¥16.12
4. 总时间：6-12小时（小说）+ 2-4小时（视频）= 8-16小时
```

#### 4.3.2 成本优化策略
- **批量处理**: 批量生成降低成本
- **缓存复用**: 相同内容缓存复用
- **资源优化**: 优化算法减少资源使用
- **错峰处理**: 在低峰期处理大任务

### 4.4 评审与重复生成成本

#### 4.4.1 评审成本
- **自动评审**: 系统自动评审，成本可忽略
- **人工评审**: 如需人工介入，按时间计费
- **修改成本**: 根据修改范围计算

#### 4.4.2 重复生成成本
- **部分重生成**: 只重生成问题章节
- **完全重生成**: 整个流程重新执行
- **优化重生成**: 基于原有结果优化
