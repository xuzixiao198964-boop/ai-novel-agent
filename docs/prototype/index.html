AI Novel Media Agent

一站式AI内容创作平台 - 原型设计文档

项目概述

项目愿景

打造一站式的AI内容创作平台，整合小说生成、视频制作、多平台发布功能，为用户提供从创意到发布的完整解决方案。

核心功能

  • AI自动生成小说（7个Agent流水线）
  • 小说转视频（5种生成模式）
  • 资讯转视频（大模型归纳总结）
  • 多平台自动发布
  • OpenClaw深度集成

技术架构

  • 前端：React/Vue + TypeScript
  • 后端：FastAPI + Python
  • 数据库：PostgreSQL + Redis
  • 部署：Docker + Nginx
  • AI模型：DeepSeek/Claude
