## 测试运行

### 运行所有测试
```bash
cd tests/frontend
node run-tests.js
```

### 运行特定测试类型
```bash
# 运行单元测试
npx vitest run unit

# 运行集成测试
npx vitest run integration

# 运行特定测试文件
npx vitest run static_contract.test.js
```

### 生成覆盖率报告
```bash
npx vitest run --coverage
```
