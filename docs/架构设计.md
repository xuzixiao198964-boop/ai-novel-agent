### 3.3 缓存策略（Redis）

| Key 模式 | 用途 | TTL |
|----------|------|-----|
| `task:progress:{task_id}` | 任务实时进度 | 实时更新，任务完成后 24h |
| `queue:position:{task_id}` | 排队位置 | 实时更新 |
| `user:session:{token}` | 用户会话 | 24h |
| `rate:api:{api_key}` | API 限流计数 | 按窗口滚动 |
| `cache:novels:public` | 公共广场小说列表 | 5min |

## 4. 接口架构

## 6. 安全架构

| 层面 | 措施 |
|------|------|
| 传输层 | HTTPS (Let's Encrypt)，API 全程 TLS |
| 认证层 | JWT + OAuth 2.0 + API Key，密码 bcrypt 哈希 |
| 授权层 | RBAC（游客/用户/开发者/管理员），资源所有权校验 |
| 数据层 | API Key AES-256/Fernet 加密存储，敏感字段脱敏 |
| 支付层 | 支付回调签名校验，幂等处理，金额服务端计算 |
| 防护层 | 图形验证码、登录失败锁定、API 限流、SQL 注入防护 |
| 审计层 | 操作日志、安全事件日志、异常告警 |

## 8. 扩展性设计

### 8.1 Agent 插件化

```python
class BaseAgent(ABC):
    """所有 Agent 的基类，新增 Agent 只需继承并实现 run()"""
    @abstractmethod
    async def run(self, task_context: TaskContext) -> AgentResult: ...

# 注册新 Agent
pipeline.register_agent("conflict_check", ConflictAgent)
pipeline.register_agent("video_review", VideoReviewAgent)
```

### 8.2 发布适配器模式

```python
class PublisherAdapter(ABC):
    """发布适配器基类"""
    @abstractmethod
    async def publish(self, content: Content, account: PlatformAccount) -> PublishResult: ...

# 新增平台只需实现适配器
class DouyinPublisher(PublisherAdapter): ...
class XiaohongshuPublisher(PublisherAdapter): ...
```

### 8.3 TTS 引擎可插拔

```python
class TTSEngine(ABC):
    @abstractmethod
    async def synthesize(self, text: str, voice_config: VoiceConfig) -> AudioSegment: ...

class OpenAITTS(TTSEngine): ...
class EdgeTTS(TTSEngine): ...
class TencentTTS(TTSEngine): ...
```
