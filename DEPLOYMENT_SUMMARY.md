- **部署方式**: 最小化部署

### 代码结构
```
/opt/ai-novel-agent/
├── backend/app/
│   ├── main.py              # 主应用文件
│   └── api/                 # API路由
├── static/
│   └── index.html           # 官方网站
└── requirements.txt         # 依赖文件
```

### 服务配置
```ini
[Unit]
Description=AI Novel Media Agent v2.0
After=network.target

[Service]
Type=simple
User=root
WorkingDirectory=/opt/ai-novel-agent/backend/app
ExecStart=/usr/bin/python3 -m uvicorn main:app --host 0.0.0.0 --port 9000
Restart=always
RestartSec=3

[Install]
WantedBy=multi-user.target
```

## 🛠️ 部署过程总结

### 成功经验
1. **最小化部署**: 在磁盘空间有限的情况下，采用最小化部署策略
2. **渐进式开发**: 先部署核心功能，再逐步扩展
3. **自动化测试**: 部署后自动验证所有API端点
4. **错误恢复**: 完善的备份和恢复机制

### 遇到的问题及解决方案
1. **磁盘空间不足** (95% → 83%)
   - 解决方案: 清理日志文件、删除旧备份、移除venv目录
   
2. **Python包管理限制**
   - 解决方案: 使用系统包管理器或虚拟环境
   
3. **网络连接稳定性**
   - 解决方案: 增加超时设置、重试机制
