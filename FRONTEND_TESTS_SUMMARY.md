### 🚀 测试运行方式

#### 1. 运行所有测试
```bash
cd tests/frontend
npx vitest run
```

#### 2. 运行特定类型测试
```bash
# 运行单元测试
npx vitest run unit

# 运行集成测试
npx vitest run integration

# 运行契约测试
npx vitest run static_contract.test.js
```

#### 3. 生成覆盖率报告
```bash
npx vitest run --coverage
```

#### 4. 使用自定义脚本
```bash
node run-tests.js
```
